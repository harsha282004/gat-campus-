# GAT Campus Compass AI

Your AI-powered campus companion — **Navigate. Explore. Discover.**

A Next.js 14 full-stack demo showcasing an intelligent campus navigation platform with virtual tours, AI chat assistant, admissions portal, and student dashboard.

## Quick Start

```powershell
cd gat-campus-compass
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

Or run the setup script:

```powershell
.\setup.ps1
npm run dev
```

## Build

```powershell
npm run build
npm start
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero, stats, AI features, testimonials |
| `/ai-assistant` | Simulated AI chat with suggested prompts |
| `/virtual-tour` | Interactive campus tour with map & carousel |
| `/admissions` | Courses, fees, eligibility, application form |
| `/academics` | Departments, faculty, labs, calendar |
| `/facilities` | Campus facilities masonry grid |
| `/events` | Events with filters and registration |
| `/contact` | Contact form + campus map |
| `/dashboard` | Student dashboard with stats & recommendations |

## Tech Stack

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS** + custom navy/gold design system
- **shadcn/ui** + Radix UI primitives
- **Framer Motion** animations
- **Embla Carousel** for testimonials & tour
- **React Hook Form** + Zod validation
- **next-themes** dark mode (default: dark)
